MOTHERBOARD

¿What it is used for?

The motherboard is the main circuit board of a computer that connects and allows communication between all the critical components of a system. It serves as the backbone that houses the CPU, memory, expansion cards, and provides connectivity for all peripherals.

How it works:

It contains circuits, chipsets, and buses that transfer data between components. The motherboard's chipset determines what components are compatible and how they communicate. It also manages power distribution to all connected components.

Key specifications to consider:

  • Form Factor: ATX, Micro-ATX, Mini-ITX (determines size and compatibility with cases)
  • Socket Type: Determines which CPU series is compatible (e.g., AM5 for AMD, LGA 1700 for Intel)
  • Chipset: Determines features, overclocking capabilities, and expansion options
  • Memory Support: Maximum RAM capacity and supported speeds
  • Expansion Slots: Number and type of PCIe slots for graphics cards and other expansions

PROCESSOR (CPU)

¿What it is used for?

The CPU (Central Processing Unit) is the primary component that executes instructions and processes data. It's often called the "brain" of the computer, handling calculations, logic operations, and controlling the flow of data throughout the system.

How it works:

The CPU fetches instructions from memory, decodes them, executes the operations, and then stores the results. Modern CPUs contain multiple cores that can process instructions simultaneously, and many include integrated graphics processors for basic display functions.

Key specifications to consider:

  • Cores/Threads: More cores allow for better multitasking and performance in parallel workloads
  • Clock Speed: Measured in GHz, higher speeds generally mean better performance in single-threaded tasks
  • Cache: Faster memory built into the CPU that improves performance
  • TDP (Thermal Design Power): Indicates power consumption and cooling requirements
  • Architecture: Newer architectures typically offer better performance and efficiency

RAM

¿What it is used for?

RAM (Random Access Memory) is the temporary, high-speed storage that holds data the CPU is actively using. It provides quick access to information that programs need to function, significantly improving system performance compared to retrieving data from storage drives.

How it works:

Unlike storage drives, RAM is volatile memory that loses its data when power is turned off. It serves as a workspace for the CPU, temporarily storing program code, operating system functions, and data being processed. More RAM allows for more programs to run simultaneously without slowdowns.

Key specifications to consider:

  • Capacity: Measured in GB, more capacity allows for more programs to run simultaneously (16GB is standard for gaming, 32GB+ for professional work)
  • Speed: Measured in MHz, higher speeds provide faster data transfer (DDR4: 2400-4800MHz, DDR5: 4800-7200MHz+)
  • Latency: Lower CAS latency (CL) numbers indicate better performance
  • Type: DDR4 vs. DDR5 (newer DDR5 offers higher bandwidth but with higher latency)
  • Channels: Dual or quad-channel configurations improve performance

STORAGE

¿What it is used for?

Storage devices permanently store all your data, including the operating system, programs, documents, photos, videos, and games. Unlike RAM, storage retains information even when the computer is powered off, providing long-term data preservation.

How it works:

There are several types of storage technologies. HDDs (Hard Disk Drives) use magnetic platters and mechanical read/write heads, while SSDs (Solid State Drives) use flash memory chips with no moving parts. NVMe SSDs connect directly to the PCIe bus for even faster speeds than SATA SSDs.

Key specifications to consider:

  • Type: HDD (capacity-focused), SATA SSD (balanced), NVMe SSD (performance-focused)
  • Capacity: Measured in GB or TB, determines how much data can be stored
  • Speed: Read/write speeds measured in MB/s or GB/s
  • Interface: SATA (up to 600MB/s), PCIe 3.0 (up to 3500MB/s), PCIe 4.0 (up to 7000MB/s), PCIe 5.0 (up to 14000MB/s)
  • Endurance: Measured in TBW (Terabytes Written), indicates lifespan

GRAPHICS CARD (GPU)

¿What it is used for?

The graphics card (GPU) is responsible for rendering images, videos, and animations to display on your monitor. It handles the complex calculations needed for 3D graphics in games, video editing, 3D modeling, and other visually intensive tasks, offloading this work from the CPU.

How it works:

Modern GPUs contain thousands of small processing cores working in parallel to render graphics. They have dedicated video memory (VRAM) to store textures, frame buffers, and other graphical data. GPUs also accelerate tasks like video encoding/decoding, AI processing, and scientific calculations.

Key specifications to consider:

  • GPU Architecture: Newer architectures offer better performance and features
  • VRAM: Amount and type (GDDR6, GDDR6X) affects performance with high-resolution textures and multiple monitors
  • Core Count: More CUDA cores (NVIDIA) or Stream Processors (AMD) generally mean better performance
  • Clock Speed: Base and boost clocks affect performance
  • Power Requirements: TDP rating and recommended PSU wattage
  • Ray Tracing/AI Capabilities: Special hardware for realistic lighting and AI-enhanced features

COOLING SYSTEM

¿What it is used for?

Cooling systems remove heat generated by computer components, particularly the CPU and GPU, to prevent thermal throttling and potential damage. Effective cooling ensures optimal performance, stability, and longevity of your system, especially during intensive tasks like gaming or rendering.

How it works:

There are several cooling methods: Air cooling uses heatsinks with metal fins to dissipate heat, paired with fans to move air. Liquid cooling circulates coolant through a closed loop to transfer heat from components to a radiator. Both methods work to maintain safe operating temperatures under load.

Types of cooling solutions:

  • Air Coolers: Budget-friendly, reliable, and maintenance-free. Range from stock coolers to large tower designs with multiple fans
  • AIO (All-In-One) Liquid Coolers: Closed-loop systems with pre-filled coolant, radiator, pump, and fans. Easier to install than custom loops
  • Custom Liquid Cooling: Advanced solution with separate components (pump, reservoir, radiators, tubing). Offers best performance but requires maintenance
  • Case Fans: Additional fans that improve airflow through the case, creating positive or negative pressure
  • Passive Cooling: Heatsinks without fans, relying solely on natural convection (used in low-power systems)

POWER SUPPLY UNIT (PSU)

¿What it is used for?

The Power Supply Unit (PSU) converts AC power from your wall outlet into the DC power that computer components require. It provides stable, clean power at various voltage levels (12V, 5V, 3.3V) to all components in your system, ensuring reliable operation and protecting against power fluctuations.

How it works:

The PSU transforms high-voltage alternating current into lower-voltage direct current through a series of transformers, capacitors, and voltage regulators. Modern PSUs use switching technology for higher efficiency. They also include protection circuits against overload, overvoltage, and short circuits.

Key specifications to consider:

  • Wattage: Total power output capacity (typically 550-850W for gaming PCs, 1000W+ for high-end builds)
  • Efficiency Rating: 80 PLUS certification (Bronze, Silver, Gold, Platinum, Titanium) indicates energy efficiency
  • Modularity: Non-modular, semi-modular, or fully modular cable management
  • Rail Design: Single +12V rail vs. multiple rails for power distribution
  • Form Factor: ATX, SFX (for small form factor builds), etc.
  • Protection Features: OVP, UVP, OCP, OPP, SCP, OTP for component safety

CABINET (TOWER OR CASE)

¿What it is used for?

The computer case (cabinet or tower) houses and protects all components while providing proper airflow for cooling. Beyond its functional purpose, the case also defines the aesthetic of your build, available space for components, and potential for future upgrades.

How it works:

Cases are designed with mounting points for the motherboard, drive bays for storage, slots for expansion cards, and spaces for fans or radiators. Modern cases focus on cable management, airflow optimization, and tool-less installation. They often include dust filters to maintain a clean internal environment.

Key specifications to consider:

  • Form Factor: Full Tower, Mid Tower, Mini Tower, Small Form Factor (determines size and component compatibility)
  • Motherboard Support: E-ATX, ATX, Micro-ATX, Mini-ITX compatibility
  • Expansion Slots: Number of PCI slots available for graphics cards and other expansions
  • Drive Bays: Number and type of bays for 2.5" SSDs and 3.5" HDDs
  • Cooling Options: Fan mounting points, radiator support, included fans
  • I/O Ports: Front/top panel connections (USB, audio, etc.)
  • GPU Clearance: Maximum graphics card length supported
